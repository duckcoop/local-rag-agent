Your Documents

How It Works

Drop your files (.txt, .md, .pdf, .json, .csv, .yaml, .log, .cfg, .ini) into the data/docs/ folder, then click Re-index. The agent splits documents into chunks, generates embeddings, and stores them in a FAISS vector index for fast retrieval.

Model Manager

Switch between models or download new ones. Larger models produce better answers but use more RAM and run slower. After switching, restart the web app to load the new model.
